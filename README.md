# HBBr
Hbb Renderer.
这是一个完全基于Vulkan API所编写的渲染器。  
(做着玩的,希望未来有用武之地) 

            
目前有3个配置可选：     
1.Debug             （Editor Debug，启动项：Editor）            
2.Release           （Editor Release，启动项：Editor）              
3.ReleaseGame_GLFW  （Renderer Only with GLFW,启动项：RendererCore）
